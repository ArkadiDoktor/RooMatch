# RooMatch Project Repository

## Folder Layout

- **interm_notebooks**  
  Contains Jupyter notebooks for generating synthetic room images, creating embeddings with DINOv2, and evaluating model performance using Recall@K.

- **interm_dataset**  
  Contains all dataset files:  
  - `labels.csv` with image paths and room IDs  
  - `real/` folder with original room images  
  - `synthetic/` folder with 10 generated images for each real image

## Notes

- `AI-Interior-Design-Proposal.pdf` – Project proposal document  
- `RooMatch.pdf` – Written report of the project  
- `RooMatch.pptx` – Presentation slides
